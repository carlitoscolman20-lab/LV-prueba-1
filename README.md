   Quiz Educativo 

 Descripción
Este proyecto es un quiz interactivo desarrollado en Python que pone a prueba tus conocimientos en cultura general.  
El juego presenta preguntas de distintas dificultades (Fácil, Medio y Difícil) y muestra el puntaje final al terminar.  

Cada ejecución selecciona un conjunto aleatorio de preguntas, para que la experiencia sea diferente cada vez que juegues.


  Objetivo del proyecto
El objetivo de este proyecto es fortalecer las habilidades de programación en Python, aplicando estructuras de control, funciones, listas y diccionarios.  
Además, busca fomentar el pensamiento lógico y la creatividad mediante la creación de una interfaz de texto divertida y educativa.


  Funcionamiento general
1. Al iniciar el programa, el usuario elige una dificultad:  
   -  Fácil  
   -  Medio  
   -  Difícil  

2. El juego selecciona automáticamente 10 preguntas correspondientes a la dificultad elegida.  
3. El usuario responde cada pregunta eligiendo una de las opciones.  
4. Al final, el juego muestra el resultado total, indicando cuántas respuestas fueron correctas y cuáles no.  


    Estructura del programa
El código se organiza en bloques principales:
- Definición de preguntas: listas o diccionarios con las preguntas, opciones y respuestas correctas.  
- Selección de dificultad: el usuario elige el nivel y el programa carga las preguntas adecuadas.  
- Lógica del juego: recorre las preguntas, registra las respuestas y calcula los aciertos.  
- Resultado final: muestra el puntaje y las respuestas correctas/incorrectas.  


  Cómo ejecutar el programa
 Asegúrate de tener Python 3.x instalado en tu sistema.
